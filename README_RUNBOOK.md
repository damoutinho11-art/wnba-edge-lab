# WNBA Edge Lab Clean Production Package (V21.8)

This package is the cleaned WNBA Edge Lab system. It removes old duplicate apps, experiments, `.git`, and large raw cache files.

## Main web app

```powershell
py app.py
```

For Render:

```text
Procfile: web: gunicorn app:app
```

## Main safe model cycle

```powershell
py run_v21_8_advisory_cycle.py
```

Optional modes:

```powershell
py run_v21_8_advisory_cycle.py --features
py run_v21_8_advisory_cycle.py --props
py run_v21_8_advisory_cycle.py --features --props
```

## Environment

Set the Odds API key in the active PowerShell session:

```powershell
$env:ODDS_API_KEY="YOUR_KEY_HERE"
```

Do not commit real API keys. Use `.env.example` only as a template.

## Safety

The current V21.8 system is advisory/manual-review only:

- no auto-betting
- no formula replacement
- no staking change
- no threshold change
- Hermes remains manual approval

## Core files

- `ultimate_fetcher_v21_4_1.py` — official WNBA + Odds API + SportsDataverse fetcher
- `feature_builder_v21.py` — V21 feature layer
- `feature_diagnostics_v21.py` — feature column diagnostics
- `model_upgrade_recommender_v21.py` — model change planning
- `model_backtester_v21_2.py` — validation/candidate layer

- `signal_execution_bridge_v20_1.py` — ticket/execution grouping
- `hermes_state_builder_v20_1.py` — Hermes manual approval state
- `model_audit_v20.py` — audit/health
- `run_v21_8_advisory_cycle.py` — full safe cycle
